# Cipher

**Cipher uses Claude's Vision AI to identify public figures from a photo or webcam capture, returning a full identity profile in seconds.**

AI-powered face recognition that identifies public figures in seconds. Upload a photo or use your webcam, and Claude's Vision API analyzes the face to return a full identity profile — name, biography, nationality, and confidence score. Built with vanilla JS and the Anthropic API.

---

## ✨ Key Features

- **AI face recognition via Claude Vision API
- **Upload a photo or scan live via webcam
- **Returns name, bio, nationality, tags & confidence score
- **Animated scan overlay with real-time system log
- **Zero backend; runs entirely in the browser
